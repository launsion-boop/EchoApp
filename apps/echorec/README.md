# EchoREC

EchoREC is an EchoBraid app for real-time meeting transcription, speaker diarization, rolling summarization, and AI research dispatch.

## Update Source

```json
{
  "type": "github",
  "repo": "launsion-boop/EchoApp",
  "app_path": "apps/echorec/app.json"
}
```

EchoBraid reads this source from the app metadata and checks `app.json` for newer versions.
